Getting Started

Install Python 3.10+, then run recab3.py after installing requirements via:

pip install PyQt5

Usage

  • Switch between Python, JS, or Blocks via tabs
  • Use the Project tab to save/load work
  • Ask the Recab AI questions or request code

Tips

Use monospaced fonts like JetBrains Mono for best readability. Place default templates in /lang.
